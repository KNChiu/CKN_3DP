### CKN_3DP  
## 3DP by marlin


## Firmware :  
Marlin 1.1.4  
ramps 1.6  
Arduino mega 2560  

## XYZ
X_Max : 120mm  
Y_Max : 200mm  
Z_Max : 170mm  

## Stepper motor driver  
X : TMC2100 (256 microsteps)  
Y : TMC2100 (256 microsteps)  
Z : DRV8825 (32  microsteps)  
E : A4988   (16  microsteps)
